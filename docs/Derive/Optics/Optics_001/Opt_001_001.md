# フェルマーの原理による屈折
A->B->Cの光路長Lは、光路長の定義(1-5)より、
```math
\begin{equation}
    \begin{split}
        L &= n_{1} \sqrt{(x-0)^2 + (0- \alpha)^2} + n_{2} \sqrt{(d-x)^2 + (c - 0)^2} \\
        &= n_{1} \sqrt{x^2 + \alpha^{2}} + n_{2} \sqrt{(d-x)^2 + c^2} \tag{1-6}
    \end{split}
\end{equation}
```   
これをxについて微分すると、
```math
\begin{equation}
    \begin{split}
        \frac{dL}{dx} &= n_{1} \frac{2x}{2\sqrt{x^2 + \alpha^{2}}} + n_{2} \frac{-2(d-x)}{2\sqrt{(d-x)^2 + c^2}} \\
        &= \frac{n_{1}x}{\sqrt{x^2 + \alpha^{2}}} - \frac{n_{2}(d-x)}{\sqrt{(d-x)^2 + c^2}} \tag{1-7}
    \end{split}
\end{equation}
```   
$`\frac{dL}{dx}=0`$のとき、変化がなくなる点なのもあり、$`L`$は極小となる.  
そのため、フェルマーの原理はこの$`=0`$を適用すればいいので、  
```math
\begin{equation}
    \begin{split}
        \frac{n_{1}x}{\sqrt{x^2 + \alpha^{2}}} - \frac{n_{2}(d-x)}{\sqrt{(d-x)^2 + c^2}} = 0 \tag{1-8}
    \end{split}
\end{equation}
```   
また、図より三角形を考えると、$`\sin`$の性質より以下が言える.  
```math
\begin{equation}
    \begin{split}
        \sin{\theta_{1}} = \frac{x}{\sqrt{x^2 + \alpha^{2}}} \tag{1-9}
    \end{split}
\end{equation}
```

```math
\begin{equation}
    \begin{split}
        \sin{\theta_{2}} = \frac{d-x}{\sqrt{(d-x)^2 + c^2}} \tag{1-10}
    \end{split}
\end{equation}
```
$`(1-9),(1-10)`$を$`(1-8)`$に適用すると,
```math
\begin{equation}
    \begin{split}
        n_{1} \sin{\theta_1} - n_{2}\sin{\theta_2} = 0 \\ 
        n_{1} \sin{\theta_1} = n_{2}\sin{\theta_2} \tag{1-2}
    \end{split}
\end{equation}
```
