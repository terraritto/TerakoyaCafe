# レンズの結像式
## 横倍率  
横倍率$`\beta`$は$`\beta = \frac{像サイズ}{物サイズ} = \frac{y^{\prime}}{y}`$と定義する.  
$`FB=Z, Z^{\prime} = F^{\prime}B^{\prime}`$とする.  
相似を考えると、$`\triangle FBA \sim \triangle FHH_{2}`$なので、  
```math
\begin{equation*}
    \begin{split}
        AB:HH_2 &= BF:FH \\
        y:-y^{\prime} &= -Z:-f \\
        y^{\prime}Z &= -fy \\
        \frac{y^{\prime}}{y} &= -\frac{f}{Z}
    \end{split}
\end{equation*}
``` 
また、$`\triangle F^{\prime}H^{\prime}H_{1}^{\prime} \sim \triangle F^{\prime}B^{\prime}A^{\prime}`$なので、  
```math
\begin{equation*}
    \begin{split}
        H^{\prime}F^{\prime}:F^{\prime}B^{\prime} &= H_{1}^{\prime}H^{\prime}:B^{\prime}A^{\prime} \\
        f^{\prime}:Z^{\prime} &= y:-y^{\prime} \\
        yZ^{\prime} &= -f^{\prime}y^{\prime} \\
        \frac{y^{\prime}}{y} &= -\frac{Z^{\prime}}{f^{\prime}}
    \end{split}
\end{equation*}
``` 
よって、すべてを組み合わせると横倍率は以下のように書ける.  
```math
\begin{equation}
    \begin{split}
        \beta = \frac{y^{\prime}}{y} = -\frac{f}{Z} = -\frac{Z^{\prime}}{f^{\prime}} \tag{1-27}
    \end{split}
\end{equation}
``` 
## ニュートンの式  
$`(1-27)`$を式変形すると、
```math
\begin{equation}
    \begin{split}
        -\frac{f}{Z} &= -\frac{Z^{\prime}}{f^{\prime}} \\
        ff^{\prime} &= ZZ^{\prime} \tag{1-28}
    \end{split}
\end{equation}
``` 
これがニュートンの式である.  
レンズが空気中の場合を考える、この場合焦点距離が等しいため$`f=-f^{\prime}`$と言える.  
ここで式変形をしてみる.  
```math
\begin{equation*}
    \begin{split}
        -\frac{Z^{\prime}}{f^{\prime}} &= -\frac{(Z+f)Z^{\prime}}{(Z+f)f^{\prime}}
    \end{split}
\end{equation*}
``` 
ここでニュートンの式を式変形して$`Z=\frac{ff^{\prime}}{Z^{\prime}}`$なので、これを入れると、
```math
\begin{equation*}
    \begin{split}
        -\frac{Z^{\prime}}{f^{\prime}} &= -\frac{(\frac{ff^{\prime}}{Z^{\prime}}+f)Z^{\prime}}{(Z+f)f^{\prime}} \\
        &= -\frac{ff^{\prime}+fZ^{\prime}}{(Z+f)f^{\prime}} \\
        &= -\frac{(f^{\prime}+Z^{\prime})f}{(Z+f)f^{\prime}}
    \end{split}
\end{equation*}
``` 
ここで、$`f=-f^{\prime}`$を使うと、
```math
\begin{equation*}
    \begin{split}
        -\frac{Z^{\prime}}{f^{\prime}} &= -\frac{(f^{\prime}+Z^{\prime})(-f^{\prime})}{(Z+f)f^{\prime}} \\
        &= \frac{f^{\prime}+Z^{\prime}}{Z+f}
    \end{split}
\end{equation*}
``` 
ここで、図を見ると$`s=Z+f, s^{\prime}=Z^{\prime}+f^{\prime}`$なので、これを適用すると、
```math
\begin{equation*}
    \begin{split}
        -\frac{Z^{\prime}}{f^{\prime}} &= \frac{f^{\prime}+Z^{\prime}}{Z+f} = \frac{s^{\prime}}{s}
    \end{split}
\end{equation*}
``` 
したがって、レンズが空気中の場合は$`(1-27)`$の横倍率を使うと、これは以下のようになる.  
```math
\begin{equation}
    \begin{split}
        \beta = -\frac{f}{Z} = -\frac{Z^{\prime}}{f^{\prime}} 
        = \frac{f^{\prime}+Z^{\prime}}{Z+f} = \frac{s^{\prime}}{s} \tag{1-27-1}
    \end{split}
\end{equation}
```
