# About
実装したものとか趣味とかをまとめています.  
実装に関してはRenderingとか単純なアルゴリズムとかが最近は多めです.

## 案内
* `Derive` - 数式導出のメモ
* `Diary` - 日記
* `Future` - 来年の抱負のためのまとめ
* `GameStatistics` - ゲームのグラフィックス設定の個人的まとめ
* `Hobby` - 趣味
    * `アニメ` - 見たアニメ
    * `温泉むすめ パネル集め` - 温泉むすめのパネルを撮ったものまとめ
    * `旅行` - 行ったところまとめ
    * `本` - 読んだ本と漫画のまとめ
    * `飲んだもの` - 飲んだ酒まとめ
    * `映画とかドラマとか` - 映画とかドラマとかの見たもののまとめ
* `Implementation` - 実装系
    * `実装まとめ` - 実装したもののリンク
    * `実装したいリスト` - いわゆるToDo
* `Movie` - 作った動画関係の分類
